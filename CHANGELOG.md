# سجل التغييرات - Al-Seraj Website Changelog

## [v2.0.0] - 2026-03-22

### تم إضافته
- **قسم جديد: الخدمات الشاملة** - عرض تفصيلي لجميع خدمات الشركة مع أيقونات توضيحية
  - التصميم المفاهيمي (Conceptual Design)
  - التصميم التفصيلي (Detailed Design)
  - أنظمة الإنذار والإخماد (Asenware)
  - أنظمة التكييف الصناعي (Ekingair)
  - لوحات التحكم الكهربائية

- **قسم محسّن: عن الشركة** - معلومات محدثة تتضمن:
  - سنة التأسيس: 2006
  - سنوات الخبرة: أكثر من 17 عاماً
  - الشراكات الاستراتيجية مع الشركات العالمية
  - الفريق المتخصص من المهندسين والفنيين

- **قسم جديد: رؤيتنا وقيمنا** - عرض القيم الأساسية:
  - السلامة (Safety)
  - الاستدامة (Sustainability)
  - المجتمع (Community)
  - النزاهة (Integrity)

- **قسم محسّن: لماذا تختار السراج** - أسباب واضحة للاختيار:
  - خبرة متكاملة
  - شراكات استراتيجية
  - فريق متخصص
  - جودة وتميز

- **قسم جديد: المشاريع الرائدة** - عرض المشاريع السابقة:
  - مصنع لهلال
  - شراكات مع المجلس الأعلى للأدوية
  - مشاريع متنوعة في اليمن

- **ملف README شامل** - توثيق كامل للمشروع والخدمات

### تم تحسينه
- **تحسينات الأداء:**
  - تحسين سرعة تحميل الصور
  - تحسين الكود والإزالة من الأكواد المكررة
  - تحسين استجابة الموقع على الأجهزة المختلفة

- **تحسينات SEO:**
  - تحديث Meta Tags
  - تحسين الكلمات المفتاحية
  - إضافة بيانات منظمة (Schema.org)
  - تحسين الوصف والعناوين

- **تحسينات التصميم:**
  - تحسين عرض الخدمات
  - تحسين تخطيط الأقسام
  - تحسين الألوان والتباين
  - تحسين استجابة الموقع

- **تحسينات المحتوى:**
  - تحديث معلومات الشركة
  - إضافة معلومات عن الشراكات
  - تحسين نصوص الخدمات
  - إضافة إحصائيات الشركة

### تم إصلاحه
- إصلاح مشاكل الاستجابة على الأجهزة الصغيرة
- إصلاح مشاكل التوافقية مع المتصفحات المختلفة
