# Security Policy (سياسة الأمان)

## Reporting a Vulnerability
If you discover a security vulnerability, please report it immediately.

## Current Security Status (حالة الأمان الحالية)
**Important Note:** During the recent audit (April 2026), several suspicious external links (Spam/Gambling links) were detected on the live website `asserag.com`. 
- **Action Required:** The webmaster should audit the WordPress database and plugins for unauthorized access or "SEO injection" attacks.
- **Recommendation:** Update all plugins, themes, and use a security plugin like Wordfence or Sucuri.

**ملاحظة هامة:** خلال التدقيق الأخير (أبريل 2026)، تم اكتشاف روابط خارجية مشبوهة (روابط سبام/قمار) على الموقع المباشر `asserag.com`.
- **الإجراء المطلوب:** يجب على مسؤول الموقع مراجعة قاعدة بيانات ووردبريس والإضافات للتأكد من عدم وجود اختراقات أو حقن برمجية (SEO injection).
- **التوصية:** تحديث جميع الإضافات والقوالب واستخدام إضافات أمان قوية.
